# CareerForge

**Build. Apply. Get Hired.**

A complete full-stack SaaS application designed to help professionals build resumes, create portfolios, track job applications, and prepare for interviews.

## 🚀 Features

### Free Plan
- 2 Resumes
- 3 Resume Templates
- 15 Job Applications
- Basic Interview Questions
- Basic Dashboard

### Premium Lite (₹29/month)
- Unlimited Resumes
- Unlimited Applications
- Internship Tracker
- No Watermark PDF
- Resume Analytics

### Premium Pro (₹99/month)
- Everything in Lite
- Portfolio Builder
- Premium Templates
- Advanced Interview Prep
- Career Analytics
- Portfolio Themes
- Priority Support

### Lifetime Plan (₹299 One Time)
- Unlock All Features Forever

## 🛠 Tech Stack

### Frontend
- React with Vite
- Tailwind CSS
- Framer Motion
- Axios
- Redux Toolkit

### Backend
- Node.js
- Express.js
- MongoDB with Mongoose
- JWT Authentication
- Razorpay Integration

### Database
- MongoDB

## 🚀 Getting Started

### Prerequisites
- Node.js (v16+)
- MongoDB
- npm or yarn

### Installation

#### Backend Setup
```bash
cd server
npm install
cp .env.example .env
npm run dev
```

#### Frontend Setup
```bash
cd client
npm install
npm run dev
```
